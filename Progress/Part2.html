Now that I have a table with all the variables needed, I have to perform feature selection and will use the Boruta package. The boruta algorithm adds randomness to the dataset by creating shuffled copies of the features (shadow features). Then, it trains a random forest classifier on the extended data set and applies Mean Decrease Accuracy to evaluate the importance of each feature (higher = more important). At each iteration, the algorithm checks whether the feature has a higher Z-score than the max Z-score of its shadow features and removes those features that are deemed highly unimportant. The algorithm stops when all the features are confirmed or rejected.

Next, I implement the boruta algorithm with the independent variables I want.

set.seed(600)
# doTrace = 2 reports more detailed progress of process
boruta <- Boruta(SALE.PRICE ~ num_schools + num_food + num_parks + num_subway + RESIDENTIAL.UNITS + COMMERCIAL.UNITS + LAND.SQUARE.FEET + GROSS.SQUARE.FEET + neighborhood_ord, data = manhattan_ints, doTrace = 2)
